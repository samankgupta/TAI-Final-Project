================================================================================
OUTPUT FORMAT:
================================================================================

Each stage produces JSON with:
  - stage: Stage number
  - stage_name: Human-readable name
  - input_count: Resumes at stage start
  - output_count: Resumes at stage end
  - retention_rate: % of resumes kept
  - score_stats: Min/max/mean scores
  - ranked_resumes: Array with:
    * rank: Position in ranking
    * id: Resume ID
    * score: Relevance score
    * ground_truth_decision: For evaluation

Files created:
  outputs/stage_1.json - Filtered resumes
  outputs/stage_2.json - TF-IDF rankings
  outputs/stage_3.json - Embedding rankings
  outputs/stage_4.json - Gemini rankings
  outputs/stage_5.json - Improved Gemini (with hallucination detection)
  outputs/metrics.json - Trust metrics
  outputs/failure_analysis.json - Failure cases
